# Пример 1: Эхо-сервер и клиент WebSocket

## Описание

Сервер принимает текстовые сообщения и отправляет их обратно с префиксом "Эхо:".

## Структура

- `server.py` - сервер на FastAPI с WebSocket-эндпоинтом
- `client.py` - клиент на websockets для подключения к серверу

## Запуск

### Шаг 1: Запуск сервера

```bash
uvicorn server:app --reload --port 8000
```

### Шаг 2: Запуск клиента (в отдельном терминале)

```bash
python client.py
```

## Зависимости

```
fastapi
uvicorn
websockets
```
